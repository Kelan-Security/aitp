# Security Policy

## Reporting Security Issues
If you discover a security vulnerability in AITP, please DO NOT open a public issue. Instead, report it via our security email: security@aitp.dev.

## Disclosure Policy
We follow a 90-day disclosure policy for reported vulnerabilities. We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Hardening
AITP v0.1 includes built-in defenses against:
- Replay attacks
- Prompt injection
- Malformed headers (Fuzz tested)
